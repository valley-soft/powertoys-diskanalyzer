ValleySoft Disk Analyzer is a lightning-fast, beautifully designed disk space analysis tool built natively for Windows.

Have you ever wondered where all your hard drive space disappeared to? Whether you prefer a dedicated window or a simple keystroke, Disk Analyzer instantly scans your local drives and presents a clean, interactive breakdown of your largest files and heaviest folders.

**What's New in Version 1.3.5:**
- **Added Grid Context Menus**: Right-click items to open them in File Explorer or copy their paths to your clipboard.
- **Added Editable Address Bar**: Double-click folder breadcrumbs to type or paste any directory path.
- **Restored Smooth Real-Time Scanning**: Folder grids stream progress smoothly in real-time.
- **Improved Performance & Stability**: Resolved random application crashes when scanning deep folders and prevented window freezes on startup.
- **Command Palette Navigation**: Added "Up one level" options and keyboard shortcut hints for native Command Palette users.
