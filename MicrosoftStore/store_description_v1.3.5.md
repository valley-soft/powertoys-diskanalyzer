ValleySoft Disk Analyzer is a lightning-fast, beautifully designed disk space analysis tool built natively for Windows.

Have you ever wondered where all your hard drive space disappeared to? Whether you prefer a dedicated window or a simple keystroke, Disk Analyzer instantly scans your local drives and presents a clean, interactive breakdown of your largest files and heaviest folders.

**What's New in Version 1.3.5:**
- **Dynamic Scanning Restored**: Restored smooth, continuous real-time folder population in the UI during deep-drive scans.
- **Navigation & Help Enhancements**: Polished UI transitions and improved responsiveness on the native WinUI 3 Navigation View help pages.
- **Command Palette Stability**: Resolved duplicate entry bugs and corrected indexing registry paths for the Windows Command Palette integration.
- **Performance Fixes**: General stability improvements and optimizations for scanning locked system folders.
