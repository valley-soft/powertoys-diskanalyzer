Store vs. GitHub Versions

Disk Analyzer comes in two distinct standalone packages depending on your system administration needs:

  • Microsoft Store Version: Built for standard security environments. It runs with standard permissions and does not include the "Run as Administrator" elevation capability. Once installed, simply launch it directly from your Windows Start Menu.
  • GitHub Version: Built for power users. This version explicitly includes "Run as Administrator" capabilities, allowing you to scan heavily restricted system directories and hidden caches. Download this from the GitHub Releases page.

Install via Windows Package Manager

The fastest way to install the Standalone App directly from your terminal.

Native Win 11 Integration

Keyboard-first storage analysis.

Built directly for the new Windows 11 Command Palette. Simply type `>` to invoke Disk Analyzer globally without ever leaving your keyboard.

Installation (Command Palette)

The Command Palette extension runs as a background COM Server. You must register it manually.

  • Download the latest `DiskAnalyzerExtension.zip` from GitHub Releases.
  • Extract the contents to a permanent location (e.g., `C:\Program Files\DiskAnalyzerExtension`).
  • Open PowerShell as Administrator in that folder and run:
.\DiskAnalyzerExtension.exe -RegisterProcessAsComServer

Installation (PowerToys Run)

Install the plugin directly into your PowerToys local app data directory.

  • Download `PowerToysRun-DiskAnalyzer.zip` from GitHub Releases.
  • Close PowerToys entirely (right-click the tray icon and select Exit).
  • Extract the folder into:
%LOCALAPPDATA%\Microsoft\PowerToys\PowerToys Run\Plugins
  • Start PowerToys and ensure "Disk Analyzer" is enabled in the Run settings.
